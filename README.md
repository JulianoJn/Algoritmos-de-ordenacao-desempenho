# Ordenação Dinâmica – Árvores e Grafos

## Sobre o projeto

Este programa foi desenvolvido em **linguagem C** para a disciplina de **Árvores e Grafos**.
Ele permite ordenar números inseridos pelo usuário utilizando diferentes algoritmos de ordenação.

## Algoritmos utilizados

* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort
* Shell Sort
* Heap Sort

## Como o programa funciona

1. O usuário escolhe um algoritmo no menu.
2. Informa quantos valores deseja inserir.
3. A cada valor digitado:

   * o valor é adicionado ao vetor
   * o algoritmo ordena o vetor
   * o vetor ordenado é exibido na tela

Também são mostrados:

* número de **comparações**
* número de **trocas**
